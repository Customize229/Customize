Linux

  1. Linux的运行是由基本的程序组成的，在基础程序的组合下去完成更加庞大的任务
  2. Linux文件没有Windows系统那么多的格式种类，保存文件同一为纯文本格式

SHELL

  1. shell：提供访问内核的服务，相当于外置的Linux控制台
  2. 通常shell分为两类，分别是GUI(图形界面)和CLI(命令行终端)
  3. Linux shell 采用C/S形式：client(客户端)，server(服务器)
